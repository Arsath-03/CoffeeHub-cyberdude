# Coffee Hub - Coffee Shop Landing Page
A modern and responsive coffee shop landing page built using HTML and Tailwind CSS only.
This project was created as part of my internship task to design a visually appealing and interactive landing page without using JavaScript or external frameworks.
# Project Overview
Coffee Hub is a neighborhood cafe concept website designed to showcase:
- Brand identity
- Coffee menu
- Customer reviews
- Gallery and contact details
  
The website focuses on clean UI, smooth hover interactions, modern typography, and responsive layout using Tailwind CSS utilities.
# Technologies Used
- HTML
- Tailwind CSS
# Key Features
- Fully responsive layout (Mobile, Tablet, Desktop)
- Sticky navigation bar
- Modern typography with multiple font styles
- Interactive hover effects
